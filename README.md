# Seattle Girls Trip Website

A static GitHub Pages-ready website for the Seattle girls trip itinerary.

## Files

- `index.html` — the full website content
- `styles.css` — all styling
- `script.js` — tiny packing checklist save feature
- `assets/` — local SVG illustrations

## How to publish with GitHub Pages

1. Create a new GitHub repository, such as `seattle-girls-trip`.
2. Upload all files from this folder into the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`, then save.
6. Wait for GitHub Pages to publish the site.

## How to customize photos

Replace any SVG in `assets/` with your own image using the same file name, or update the `<img src="...">` paths in `index.html`.

## Easy edits

Open `index.html` and search for any restaurant, bar, or time you want to change.
